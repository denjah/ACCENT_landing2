# Workflow: Validate Responsive

Используй для проверки адаптивности текущей версии лендинга.

## Шаги
1. Открой accent-landing.html в browser agent
2. Проверь при 375px:
   - Нет горизонтального overflow
   - Все кнопки минимум 44x44px
   - Body-текст не меньше 16px
   - Навигация адаптирована
   - Формы удобны для ввода
3. Проверь при 768px:
   - Сетка корректно перестроилась
   - Нет сломанных flex и grid
4. Проверь при 1440px:
   - Контейнер не шире 1440px
   - Строки не длиннее 70ch
5. Создай Artifact responsive-report.md:
   - Список что работает (checkmark)
   - Список что нужно исправить (с указанием брейкпоинта)
