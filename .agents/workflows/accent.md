description: ANTIGRAVITY RULES — Акцент v2 (медленный инженерный подход)
---

Мы работаем только через один чат и один артефакт за раз.
Каждый следующий шаг добавляет ровно одну новую технологию или фичу.
Сначала — чистый HTML + CSS + JS (как ваш accent_v1_engineering_grid.html).
Только когда вы полностью понимаете и говорите «готово», переходим к React-компоненту.
Все стили строятся исключительно через CSS custom properties из design-tokens.md. Никаких hardcoded цветов, отступов или шрифтов.
Код всегда читаемый, с комментариями на русском, именами переменных понятными инженеру.
Анимации и эффекты (включая mouse-grid) добавляются только после того, как базовая структура работает.
Mouse-trail / engineering grid эффект внедряется постепенно: сначала статичная сетка → потом canvas 2D → потом (если нужно) React Three Fiber.
Каждый артефакт — полностью рабочий сам по себе. Никаких TODO и «потом доделаем».
Перед каждым новым шагом я пишу план на русском: что добавляем, почему, где это будет лежать.
Вы всегда можете сказать «стоп, объясни подробнее этот кусок» или «давай вернёмся на шаг назад».
Цель — не быстрый результат, а чтобы вы понимали каждую строку и могли сами поддерживать код.

Это правило действует до конца проекта. Если я нарушу — сразу говорите.
