# Workflow: Build Section

Используй когда нужно построить очередную секцию лендинга.

## Шаги
1. Загрузи skill design-tokens и прочитай references/design-tokens.md
2. Прочитай актуальное ТЗ (файл tz.md в корне проекта если есть)
3. Используй ТОЛЬКО CSS-переменные из design-tokens — никаких hardcoded цветов
4. Тексты — точное соответствие ТЗ, без правок
5. После написания кода: открой файл в browser agent
6. Сними скриншот при ширине 375px
7. Сними скриншот при ширине 1440px
8. Проверь чеклист:
   - Нет overflow по горизонтали
   - Текст читаем на всех поверхностях
   - CTA-кнопки хорошо видны
   - Акцентный цвет только на кнопках и иконках
   - Нет запрещённых паттернов из design-system rules
9. Список найденных проблем — отдельным артефактом
