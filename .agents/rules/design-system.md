# Design System Rules — Accent Landing

## Обязательно перед любым CSS
Загружай skill design-tokens и читай references/design-tokens.md

## Цвет
- Акцент #C41E3A (--color-primary-base) — ТОЛЬКО для: CTA-кнопки, активные иконки, счётчики цифр, индикаторы ошибок
- Не более одного акцентного оттенка в пределах одного экрана
- Никаких акцентных фонов на больших площадях
- Все поверхности — только серая шкала
- Логотипы производителей: монохром --gray-600 (#868E96), при hover -> --color-primary-base

## Типографика
- Только Inter и IBM Plex Mono — никаких других шрифтов
- IBM Plex Mono обязателен для артикулов, кодов, технических таблиц
- Letter-spacing заголовков: -0.01em
- Letter-spacing меток uppercase: +0.04em
- Максимальная ширина строки основного текста: 70ch
- Минимальный размер текста: 12px

## Кнопки
- Только три типа: primary (фон --color-primary-base) / secondary (outlined) / ghost
- Радиус кнопок: 4px (--radius-sm) — строго
- Никаких градиентных кнопок
- Padding: 12px 24px, font-weight: 600

## Карточки
- Радиус: 6px (--radius-md)
- Hover: box-shadow sm->md + transform translateY(-2px)
- Запрещены цветные border-left с акцентом
- Граница: 1px solid var(--color-border-default) или только тень

## Интерактивность
- Все переходы: transition: all 0.2s ease-in-out
- Focus ring: box-shadow 0 0 0 3px rgba(196, 30, 58, 0.3)
- Disabled: opacity 0.45, cursor not-allowed, без hover
- Count-up анимации: запускать через IntersectionObserver

## Анимации
- Только fade-in и slide-up через IntersectionObserver
- Только функциональные анимации
- @media (prefers-reduced-motion: reduce) { отключить анимации }

## ЗАПРЕЩЕНО (НИКОГДА):
- Градиентные кнопки
- Акцентный фон на больших площадях
- Скругления больше 8px на кнопках и инпутах
- Тени с высокой непрозрачностью чёрного
- Более двух шрифтовых семейств
- Текст меньше 12px
- Выравнивание основного текста по центру в блоках
- Декоративные фоновые элементы (блобы, пятна, волны, орнаменты)
- Иконки в цветных кружках как декор секций
- Одинаковая высота и отступы у всех секций подряд
- Три одинаковые карточки в ряд — варьировать компоновку
- Emoji как иконки или декор
